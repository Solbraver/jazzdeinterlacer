User Manual

JazzDeinterlacer 1.0  ·  macOS 14 Sonoma or later

1. Installation

  1. Download JazzDeinterlacer-1.0.zip from the website or your Gumroad receipt.
  2. Unzip and drag JazzDeinterlacer.app to your Applications folder.
  3. Double-click to launch.
⚠️ First launch — macOS security prompt
macOS may block the app with a message saying it "cannot be opened because Apple cannot check it for malicious software." This is Gatekeeper, which flags apps distributed outside the App Store.

To open it: go to System Settings → Privacy & Security, scroll down, and click "Open Anyway". This prompt only appears once.

2. Quick Start

  1. Click Open… in the toolbar (or drag a video file onto the window).
  2. Select a processing mode in the bottom-left tab: Deinterlace, AI Deinterlace, or Inverse Telecine.
  3. Choose your algorithm and settings.
  4. Click Preview 3s to render a short clip and watch it inline.
  5. When the result looks right, click Render to export the full file.

The output file is automatically placed in the same folder as the source, with a descriptive suffix such as _deinterlace_qtgmc_lk_59.94p.mov.

4. Processing Modes

Deinterlace

Converts interlaced video (fields) to progressive frames.

AlgorithmQualitySpeedBest for
Field BlendLowFastestQuick web previews
Spatial AdaptiveMediumFastGeneral purpose
Motion Comp.HighModerateMoving subjects
QTGMCHighestSlowBroadcast / archival

QTGMC options:

  • Neural Flow (RAFT): Replaces Lucas-Kanade optical flow with a neural network for better motion estimation. Higher quality, slower.
  • Neural Engine: Runs RAFT on Apple's Neural Engine via CoreML. Faster on Apple Silicon.

Field Order controls which field is treated as temporally first.

  • Auto — reads the flag from the source file. Correct for most broadcast footage.
  • Lower / Upper Field First — override when source metadata is wrong (visible as a "venetian blind" combing pattern).

Output Rate:

  • Same as source — one progressive frame per input field pair. Halves the frame rate.
  • Double rate — one progressive frame per field. Preserves the original temporal resolution; doubles the frame rate. Recommended for motion-heavy content.

AI Deinterlace

Uses Neural QTGMC — RAFT optical flow combined with QTGMC temporal smoothing, running on Apple Neural Engine. This is the highest quality mode available.

Trial: Every download includes 2 free AI Deinterlace exports. After the trial, a license ($4.99 one-time) is required to continue using AI Deinterlace. All other features remain free forever.

Enable Neural Engine (CoreML/ANE) to offload RAFT inference to the dedicated ANE chip. Strongly recommended on Apple Silicon. First run is slower due to CoreML model compilation; subsequent runs are fast.

Inverse Telecine (IVTC)

Removes 3:2 pulldown from film-sourced content that was telecined to 29.97i. Output is always 23.976p (or 24p with the conform option).

On file open, JazzDeinterlacer automatically scans the first 10 seconds to detect the pulldown cadence. The detected phase (A–E) and confidence percentage are shown in the settings panel.

  • Phase override: If auto-detection is wrong (visible as residual combing), manually select the correct phase with the A/B/C/D/E buttons.
  • Conform 23.976 → 24: Speeds up output by 0.1% to match true 24p. Audio pitch is corrected automatically. Use when delivering to a 24p timeline.
Note: The PROCESSED panel shows a placeholder during scrubbing — IVTC needs sequential frames and can't show a meaningful single-frame preview. Use Preview 3s or Render to see the actual output.

5. Colour Controls

The Colour section (middle column of the bottom panel) applies post-processing adjustments to the exported file.

ControlRangeDefaultPurpose
Input RangeVideo / FullAuto-detectedDeclares the luminance range of the source
Output RangeVideo / FullMatches inputConverts range on export if needed
Gamma0.1 – 4.01.00Midtone brightness curve
Brightness−0.5 – +0.50.00Global lift / lower
Contrast0.0 – 3.01.00Contrast around midpoint
Saturation0.0 – 3.01.00Colour intensity

Video vs Full range: Video range (broadcast/limited) uses luma 16–235. Full range uses 0–255. JazzDeinterlacer auto-detects the input range on file open. If it guesses wrong, correct it with the Input Range picker. Set Output Range differently only if you need a conversion (e.g. deliver full-range to YouTube from broadcast-range source).

The Reset button appears when any value differs from default and restores all controls to identity.

Output file: Automatically set to the source file's folder with a descriptive suffix. Click Change… to pick a different path.

Preview 3s: Renders only the first 3 seconds starting from the current scrub position. The result plays inline in the PROCESSED panel. Use Space to play/pause, ← → to step frames.

Render: Exports the full file. A progress bar shows frames processed. Click Cancel to stop.

7. Preview & Scopes

Scrub bar

Drag the slider or use keyboard shortcuts to move through the source. The original and processed frames update live.

1:1 Zoom

Click 1:1 to view the video at native pixel size. Drag to pan. Click Fit to return to fit-to-window mode.

Scopes window

Click the waveform icon (⌇) in the toolbar to open the floating Scopes window:

  • Waveform — luminance levels, left-to-right matches left-to-right of image.
  • Vectorscope — chrominance plotted on a circle. Distance from centre = saturation; angle = hue.
  • Histogram — per-channel (R, G, B) pixel value distribution.

Scopes update on every scrub frame and can display either the Input (original) or Output (processed) signal — toggle with the selector inside the scopes window.

9. License Activation

AI Deinterlace is unlocked with a license key purchased at Gumroad.

  1. Open JazzDeinterlacer → Settings (⌘,), or click Activate License in the paywall sheet.
  2. Paste your license key.
  3. Click Activate.

The app validates the key against Gumroad's servers. An internet connection is required for initial activation. After that, the license is stored in the macOS Keychain — no connection needed to run the app.

One license covers 2 Mac activations. To move to a new Mac, contact support via the website.

10. Troubleshooting

"JazzDeinterlacer can't be opened because Apple cannot check it for malicious software."

Open System Settings → Privacy & Security and click Open Anyway. This appears once and is expected for apps distributed outside the App Store.

The PROCESSED panel shows a placeholder for IVTC.

IVTC needs sequential frames — single-frame preview isn't meaningful. Use Preview 3s or Render to see the actual output.

Output still has combing / looks wrong.

Check the Field Order setting. If Auto is wrong, toggle between Lower Field First and Upper Field First. For IVTC, try overriding the detected phase with the A/B/C/D/E buttons.

H.264 / H.265 is greyed out or shows an FFmpeg warning.

The bundled FFmpeg binary may have lost its execute permissions. Re-download the app from the website.

AI Deinterlace is slow on first use.

CoreML compiles the RAFT model on the first run — this is a one-time cost. Enable Neural Engine in the AI Deinterlace settings for maximum speed on Apple Silicon.

"License validation failed."

Check your internet connection. The key field trims whitespace automatically, but double-check for invisible characters if you typed it manually. Contact support via the feedback form if the problem persists.
